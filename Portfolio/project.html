Notes App

Android Studio . Xml . Firebase . Figma

Notes app is my personal project to learn and explore the field of android development. Project Details: Project Name: Notes App Objective: To create a simple notes app allowing users to create, edit, and delete notes, with features like login, email verification, and cloud backup. Technologies Used: Android Studio, Firebase (for authentication and cloud backup), XML for UI design. My Role: Project Developer: I was responsible for designing, developing, and implementing the entire project. UI/UX Designer: Designed the user interface using XML layouts, ensuring a user-friendly experience. Backend Integration: Integrated Firebase Authentication for user login and email verification, and Firebase Realtime Database for cloud backup of notes data. Testing and Debugging: Conducted thorough testing to ensure the app functioned correctly and addressed any bugs or issues. Outcome: Functional Notes App: Successfully developed a fully functional notes app allowing users to create, edit, and delete notes. Secure User Authentication: Implemented secure email-based login and email verification using Firebase Authentication. Cloud Backup: Enabled automatic cloud backup of notes data using Firebase Realtime Database, ensuring data persistence and accessibility across devices. User-friendly Interface: Designed a visually appealing and intuitive user interface, enhancing the overall user experience. Valuable Learning Experience: Gained valuable experience in Android development, UI/UX design, and backend integration, furthering my skills and knowledge in the field. I take the help from you tube and other some learning platform to know all the features because i am totally new for android development field.
